updateLog

PURPOSE

updateLog updates the console log in the GridInspector Window.

SYNOPSIS

function updateLog( log, strin, multiline )

DESCRIPTION 

updateLog updates the console log in the GridInspector Window.

   @desc Updates the console log in the GridInspector Window.

%--------------------------------------------------------------------------------------------------
